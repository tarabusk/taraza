== Installation ==

To setup the Home Page and option design : Go to Appearance> taraza Options

You can use a Contact Form Page Template, and settle the contact email in the Administration taraza Options.

For a better nice looking Home Page : Go to Settings >> Reading and select a number of posts displayed as a mutliple of 3.
